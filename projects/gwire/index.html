gwire

gwire is a p2p, encrypted messaging application that uses RSA with 2048 bit keys and 256 bit AES to ensure safe communication. It works by opening a selected (range of) port(s) on a host to which the clients can connect to. Each client has 1 connection and the host has n connections for n hosts.
