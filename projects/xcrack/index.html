XCrack

XCrack is a hash cracking tool, designed for brute force and wordlist attacks. To tool is designed to use wordlists or brute force to crack a hash. Currently supported hash types are: MD5, SHA1, SHA256. More will be added soon. It can also generate wirdlist from preferences like characters, length or based on other wordlists.
